## Default Permission

Default permissions for the hdmv plugin

#### This default permission set includes the following:

- `allow-hdmv-open-disc`
- `allow-hdmv-close-disc`
- `allow-hdmv-get-disc-info`
- `allow-hdmv-list-titles`
- `allow-hdmv-list-playlists`
- `allow-hdmv-get-playlist`
- `allow-hdmv-start-navigation`
- `allow-hdmv-send-key`
- `allow-hdmv-mouse-move`
- `allow-hdmv-mouse-click`
- `allow-hdmv-render-preview`
- `allow-hdmv-get-menu-state`
- `allow-hdmv-build-disc`
